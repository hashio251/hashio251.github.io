# マイポートフォリオサイト

## 概要
このサイトは、私のスキルや作品の紹介サイトです。
IT・デザイン・プログラミングの学習成果をまとめたサイトです。



## 作品について
### コンセプト
ホーム写真に合わせ落ち着き・自然・安心感など印象ある緑色をベースに構成しています。


### 主要な要素
- **home**: サイトホーム画面
- **about**: 経歴や使用言語について（自己紹介スライドのパワーポイント有り）
- **production**: サイト作品（HTML・CSS・JSを使用したサイト）
- **gallery**: OpenProcessing・figma・Photoshopなどの作品
- **contact**: コンタクトページ



## サイトの特徴
### デザイン
- **ナビゲーションデザイン**: OSSより引用（色や文字やサイズなど改変）
- **レスポンシブデザイン**: スマートフォンやタブレットにも対応
- **スムースアニメーション**: スクロールやホバー時の円滑な動き

### 機能
- **ナビゲーション**: 固定ヘッダー
- **パーツ紹介**: 使用した各パーツの詳細・手順説明
- **ギャラリー**: 作品の様々な角度からの紹介



## ディレクトリ構成
```
├── index.html          # メインHTMLファイル
├── styles.css          # メインスタイルシート
├── about.html          # アバウトHTMLファイル
├── about.css           # アバウトスタイルシート
├── production.html     # プロダクションHTMLファイル
├── production.css      # プロダクションスタイルシート
├── gallery.html        # ギャラリーHTMLファイル
├── gallery.css         # ギャラリースタイルシート
├── contact.html        # コンタクトHTMLファイル
└── contact.css         # コンタクトスタイルシート
```



## 使用方法
1. `index.html` をブラウザで開く
2. ナビゲーションメニューから各セクションに移動
3. 各セクションに配置している画像に埋め込んだ外部リンク（ブランクあり）から自己紹介スライドや作品ページに遷移



## 技術仕様
- **HTML5**:文字など
- **CSS3**:キーフレームを使用したりレイアウト調整など
